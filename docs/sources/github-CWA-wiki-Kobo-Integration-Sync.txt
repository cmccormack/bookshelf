## Device Configuration File

On the Kobo, locate the .kobo/Kobo/ folder and edit "Kobo eReader.conf" in a text editor.
The [OneStoreServices] section contains the key setting:

  api_endpoint=https://yourserver.example/kobo/<your-auth-token>

This single line redirects the device's library requests to the CWA server instead of
Kobo's default services.

## Authentication Token

Access CWA from a non-localhost address to get a valid endpoint. Generate the token
via user account settings; the system displays an api_endpoint URL to copy into the conf file.

## Firmware-Specific Issue (Newer Devices)

Recent Kobo devices (such as Libra Colour) automatically restore the configuration file,
potentially erasing custom api_endpoint entries on sync.

For these models, alternative settings can be used:
  image_host: The server address
  image_url_quality_template: Template for cover images
  image_url_template: Standard image template

## Post-Configuration

After editing: safely eject device, restart, connect to WiFi. Initial sync begins
automatically. Books from CWA library should appear in the device's library.
