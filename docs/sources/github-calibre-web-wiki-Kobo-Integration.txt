The Kobo eReader.conf file is located in the .kobo/Kobo directory on Kobo devices.
This configuration file determines which URL the device uses for syncing books with Calibre-Web.

Default API endpoint in [OneStoreServices]:
  api_endpoint=https://storeapi.kobo.com

Users can generate a custom synchronization URL through Calibre-Web by clicking the
"Create/View" button under their Calibre-Web profile page.

The custom endpoint redirects the device's library requests to the CWA server instead
of the official store.

Reverse proxy: may need to change api_endpoint from http:// to https://, and force
the header: proxy_set_header X-Scheme https;

Only EPUB formatted books can be synchronised. PDFs are not supported due to
device restrictions.
