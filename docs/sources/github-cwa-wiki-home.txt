# Calibre-Web-Automated Wiki - Home

**Welcome to the Calibre-Web-Automated Wiki!**

The wiki serves as the official documentation hub for Calibre-Web-Automated (CWA), built atop the Calibre-Web project. The introduction acknowledges that the wiki remains under development and welcomes community contributions through Discord, GitHub issues, or direct edits.

**Key sections include:**

For end users, documentation covers configuration, authentication methods (including OAuth and LDAP), Kobo integration, metadata fetching, and password reset procedures.

For developers, resources address custom Docker image creation, contributor testing guidelines, and translation contributions.

**Critical Network Deployment Warning:**

The documentation emphasizes a significant limitation: "SQLite is designed as a lightweight, file-based database system, and it assumes the underlying file system supports certain guarantees about file locking, atomic writes, and consistency." Network-based deployments—particularly using NFS—can create unpredictable issues.

Starting with version 3.0.0, such deployments are officially unsupported. Users may attempt network-share installations at their own discretion, but the project cannot provide troubleshooting assistance for resulting complications.

The resource directs users to Calibre-Web's documentation for core functionality questions not addressed in the CWA-specific wiki.
