## eReader Configuration Steps

1. Connect via USB and navigate to .kobo\Kobo folder
2. Backup: duplicate Kobo.eReader.conf as Kobo.eReader.conf.backup
3. Open Kobo.eReader.conf with a text editor
4. Find the [OneStoreServices] section
5. Replace the line starting with api_endpoint=https://storeapi.kobo.com with the Calibre-Web token endpoint
6. Save and safely eject the device

File path: .kobo\Kobo\Kobo.eReader.conf (Windows backslash notation)
Section: [OneStoreServices]
Key: api_endpoint=

Note: Backup allows easy restoration by renaming .backup file back to original name.

Important Limitations:
- Only EPUB formatted books can be synchronised
- No mention of Beta Features menu
- No firmware-specific version notes
- HTTP and HTTPS cannot share the same port number
